iPhone / iPad / Mac

  1. 打开 设置 → 日历 → 账户 → 添加账户 → 其他 → 添加订阅日历
  2. 粘贴上面的链接，点"订阅"

Google Calendar（网页版）

  1. 打开 calendar.google.com
  2. 左侧 "其他日历" 旁边点 + → 通过网址添加
  3. 粘贴上面的链接

Outlook 网页版

  1. 打开 Outlook 日历 → 添加日历 → 从 Internet 订阅
  2. 粘贴上面的链接

安卓

  1. 打开 Google Calendar App → 左上角菜单 → 设置 → 添加日历 → 通过网址添加
